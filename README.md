# CodeBean Backend

The backend repository for _CodeBean_.

## Package Information

- Kotlin 1.6.10 with JVM 11
- Gradle 7.4.1
- Spring Boot 2.6.6
- Spring Data JPA
- Querydsl 5.0.0
- JUnit 5, AssertJ, Mockito
- Spring REST Docs

## Setup & Run

### 1. Build with Gradle

```bash
$ ./gradlew build
```

### 2. Run the server with .jar

```bash
$ java -jar ./build/libs/codebean-0.0.1-SNAPSHOT.jar
```
